# Metrolist-to-diffuse-bridge
A stability bridge for [metrolist] to fix album art and color synchronization issues.

## YT video on how to get it working:
https://youtu.be/Teo0L9CO_eU?si=QCoVU4ns4lZe151n

## Why this exists
[Metrolist] often struggles to report its playback state to Android, causing:
* Persistent "ghost" notifications.
* Frozen album art/colors on the lock screen.
* Unnecessary battery drain from background "zombie" processes.

## Features
* **Color Fix:** Properly forces the system to recognize the current track to pull album colors.
* **System Stability:** Cleans up notification sessions automatically to stop background drains.
* **Theme Support:** Includes built-in Light/Dark mode.

## ⚠️ Important Note
This app specifically addresses synchronization issues for album art and colors. 
**(Note: this app only fixes the album colour not showing up but it doesn't fix the beats.)**
